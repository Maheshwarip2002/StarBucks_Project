**☕ Starbucks Sales Performance Analysis — Power BI Dashboard**

Interactive Power BI dashboard analyzing Starbucks sales performance — revenue trends, product category breakdown, and regional insights.

**Objective**
The objective of this project is to analyze Starbucks sales and performance data using Power BI to uncover meaningful business insights. The dashboard visualizes key metrics such as revenue trends, product category performance, regional sales distribution, and customer purchasing patterns to support data-driven decision-making.


**Tools & Technologies Used**

- **Power BI Desktop** — data modeling, visualization, and dashboard creation
- **Power Query** — data cleaning and transformation
- **DAX (Data Analysis Expressions)** — calculated measures and KPIs
- **Excel/CSV** — source data format


**Data Source**

The dataset includes Starbucks sales records with details such as order dates, product categories, quantities sold, revenue, and store locations.

**Methodology**

1. **Data Collection** — Gathered raw Starbucks sales data
2. **Data Cleaning** — Used Power Query to remove duplicates, handle missing values, and format columns correctly
3. **Data Modeling** — Built relationships between tables (Sales, Products, Stores, Dates)
4. **DAX Measures** — Created calculated fields such as Total Revenue, Average Order Value, and Growth %
5. **Visualization** — Built interactive charts, maps, and KPI cards
6. **Dashboard Design** — Arranged visuals into a clean, easy-to-navigate report layout


**Dashboard Features**
- Sales trend analysis over time (monthly/yearly)
- Revenue breakdown by product category
- Regional/store-wise performance comparison
- Top-selling products and categories
- Interactive filters and slicers for dynamic exploration

**Key Insights**

- Beverages contributed the highest share of total revenue compared to food items
- Sales peaked during [specific month/season], indicating seasonal demand
- [Region/Store] recorded the highest overall sales performance
- Steady year-over-year growth trend observed in overall revenue


**Future Scope**

- Integrate real-time sales data using live data connections
- Add predictive analytics for sales forecasting
- Expand dashboard with customer segmentation analysis
- Publish report to Power BI Service for wider access
